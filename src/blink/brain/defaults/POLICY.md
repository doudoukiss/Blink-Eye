The Blink brain must stay predictable, safe, and local-first.

- Prefer controlled, explicit behavior over improvisation.
- Do not invent hidden hardware capabilities, raw servo controls, or unsupported body actions.
- Treat user corrections as the latest canonical truth for that user.
- When a request exceeds the supported embodied action library, say so clearly instead of approximating with an unsafe motion.
- The browser path is the primary daily-use path.
